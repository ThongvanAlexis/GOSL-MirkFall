**Per-tile paint pattern** (reused across all 4 variants via `buildUnrevealedCellsPath`):
```dart
// For each VisibleMirkTile in context.visibleTiles:
//   1. Compute tile screen rectangle from tile lat/lon + viewport + size
//   2. For each cell (i, j) in the 64û64 bitmap:
//      - If bit is 0 (un-revealed) ã include cell rect in combined Path
//      - If bit is 1 (revealed) ã skip
//   3. Use a single Path accumulated across all unrevealed cells of the
//      tile, then drawPath once per tile. Far cheaper than drawRect per cell.
```

**Feather edge** (atmospheric / candlelight / heavenly): apply `MaskFilter.blur(BlurStyle.inner, sigma)` to the Paint. Sigma = cellSize û kFeatherRadiusFraction û pixelRatio.

**Noise modulation** (atmospheric / candlelight / heavenly): sample `SimplexNoise2D` at low-frequency per tile ã modulate alpha by ôÝ3-10% around baseline. Pass `context.sessionElapsed.inMilliseconds / 1000.0 * noiseSpeed` as the time coord (research consolidated on `sessionElapsed`).

**Solid**: skip noise + blur entirely. Single Paint with `color = Color.fromARGB(...)` encoding `config.colorArgb` + `config.baselineAlpha`. One `drawPath(combinedUnrevealedPath)` per tile.

**Candlelight**: radial gradient centred on `context.currentFix` (if present) projected to screen space via `MirkProjection.latLonToScreen`. Warm centre ã darker periphery colours.

**Heavenly clouds**: noise with `driftDirectionDeg = 45` ã drift vector `(cos(45ô¯), -sin(45ô¯))` (Y inverted because screen-Y grows south). Lighter baseline alpha.

```dart
import 'dart:ui' show Offset, Size;
import '../../domain/mirk/mirk_viewport_bbox.dart';

/// Projects lat/lon to a screen offset for the current viewport + canvas size.
///
/// Linear Mercator within the viewport bbox ã sufficient for the fog
/// overlay because the underlying MapLibre canvas does its own web-mercator
/// projection at the platform layer. Here we just need "put this cell
/// rectangle on the screen where MapLibre has it".
class MirkProjection {
  const MirkProjection._();

  /// Returns the screen offset for ([lat], [lon]) given the current
  /// [viewport] and canvas [size]. Offset (0,0) = top-left of canvas.
  ///
  /// Coordinates outside the viewport return offsets outside the canvas ã
  /// the caller's drawing primitives (CustomPainter, Canvas clip) handle
  /// off-screen rendering natively.
  static Offset latLonToScreen({
    required double lat,
    required double lon,
    required MirkViewportBbox viewport,
    required Size size,
  }) {
    final dLat = viewport.north - viewport.south;
    final dLon = viewport.east - viewport.west;
    // guard against zero-size bbox (should never happen in prod)
    if (dLat == 0 || dLon == 0) return Offset.zero;
    final x = ((lon - viewport.west) / dLon) * size.width;
    final y = ((viewport.north - lat) / dLat) * size.height;
    return Offset(x, y);
  }
}
```

```dart
/// Iterates UN-revealed cells of a [VisibleMirkTile], yielding a `Path`
/// containing a rectangle per unrevealed cell ã cheaper to `drawPath`
/// once per tile than `drawRect` per cell. Shared across the 4 concrete
/// renderers.
Path buildUnrevealedCellsPath({
  required VisibleMirkTile tile,
  required MirkViewportBbox viewport,
  required ({double width, double height}) canvasSize,  // avoid dart:ui Size in dartdoc
}) {
  final path = Path();
  final cellLatSpan = (tile.tileNorthLat - tile.tileSouthLat) / 64.0;
  final cellLonSpan = (tile.tileEastLon - tile.tileWestLon) / 64.0;
  for (var j = 0; j < 64; j++) {
    final cellNorthLat = tile.tileNorthLat - j * cellLatSpan;
    final cellSouthLat = tile.tileNorthLat - (j + 1) * cellLatSpan;
    for (var i = 0; i < 64; i++) {
      final bitIndex = j * 64 + i;
      final byteIndex = bitIndex >> 3;
      final bitOffset = bitIndex & 7;
      final bit = (tile.bitmap[byteIndex] >> bitOffset) & 1;
      if (bit == 1) continue;  // revealed ã skip
      final cellWestLon = tile.tileWestLon + i * cellLonSpan;
      final cellEastLon = tile.tileWestLon + (i + 1) * cellLonSpan;
      // ... use MirkProjection.latLonToScreen to get nw + se Offsets
      // ... path.addRect(Rect.fromPoints(nw, se));
    }
  }
  return path;
```

```dart
  late final Paint _paint = Paint()

      (config.baselineAlpha * 255).round(),
```

```dart
class AtmosphericMirkRenderer implements MirkRenderer {
  AtmosphericMirkRenderer(this.config, {int seed = 42})
      : _noise = SimplexNoise2D(seed: seed);
  final AtmosphericConfig config;
  final SimplexNoise2D _noise;
  bool _disposed = false;

  @override
  void paint(Canvas canvas, Size size, MirkPaintContext context) {
    if (_disposed) return;
    final tSec = context.sessionElapsed.inMilliseconds / 1000.0;
    final driftX = math.cos(config.driftDirectionDeg * math.pi / 180.0);
    final driftY = -math.sin(config.driftDirectionDeg * math.pi / 180.0);

    for (final tile in context.visibleTiles) {
      final noiseSample = _noise.noise2(
        tile.parentX * config.noiseScale + tSec * config.noiseSpeed * driftX,
        tile.parentY * config.noiseScale + tSec * config.noiseSpeed * driftY,
      );
      final alpha = (config.densityBaselineAlpha + noiseSample * 0.03).clamp(0.0, 1.0);
      final r = (config.baseColorArgb >> 16) & 0xFF;
      final g = (config.baseColorArgb >> 8) & 0xFF;
      final b = config.baseColorArgb & 0xFF;

      final featherSigma =
          (size.height / 64.0) * config.featherRadiusFraction * context.pixelRatio;
      final paint = Paint()
        ..color = Color.fromARGB((alpha * 255).round(), r, g, b)
        ..style = PaintingStyle.fill
        ..maskFilter = MaskFilter.blur(BlurStyle.inner, featherSigma);

      final path = buildUnrevealedCellsPath(
          tile: tile, viewport: context.viewportBbox, canvasSize: size);
      canvas.drawPath(path, paint);
    }
  }

  @override
  void update(Duration elapsed) {}

  @override
  Future<void> dispose() async { _disposed = true; }
}
```
