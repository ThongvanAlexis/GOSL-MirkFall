<rendering_approach>
**Per 09-RESEARCH §Rendering Strategy Decision**, all 4 renderers use Flutter `CustomPainter` primitives (Canvas + Paint + Path). No FragmentShader in Phase 09.

**Per-tile paint pattern** (reused across all 4 variants):
```dart
// For each VisibleMirkTile in context.visibleTiles:
//   1. Compute tile screen rectangle from tile lat/lon + viewport + size
//   2. For each cell (i, j) in the 64×64 bitmap:
//      - If bit is 0 (un-revealed) → draw fog at cell screen rect
//      - If bit is 1 (revealed) → skip
//   3. Use a single Path accumulated across all unrevealed cells of the
//      tile, then drawPath once per tile. Far cheaper than drawRect per cell.
```

**Feather edge** (atmospheric / candlelight / heavenly): apply `MaskFilter.blur(BlurStyle.inner, sigma)` to the Paint. Sigma = cellSize × kFeatherRadiusFraction × pixelRatio.

**Noise modulation** (atmospheric / candlelight / heavenly): sample `SimplexNoise2D` at low-frequency per tile (NOT per cell) → modulate alpha by ±5% around baseline. Pass `frameElapsed.inMilliseconds / 1000.0 * noiseSpeed` as the time coord.

**Solid**: skip noise + blur entirely. Single Paint with `color = Color(config.colorArgb).withAlpha((config.baselineAlpha * 255).toInt())`. One `drawPath(combinedUnrevealedPath)` per tile.

**Candlelight**: radial gradient centred on `currentFix` (if present) projected to screen space, using `Paint..shader = ui.Gradient.radial(...)`. Warm centre → darker periphery colours from constants.

**Heavenly clouds**: simplex noise with `driftDirectionDeg = 45` → x offset = `cos(45°) * frameElapsed`. Lighter baseline alpha.

**Projection helper** (new `lib/infrastructure/mirk/mirk_projection.dart`): lat/lon → screen pixel. Simple linear Mercator within the viewport bbox. Reusable across all 4 renderers. Wave 3 adds this file.
</rendering_approach>

**Add `lib/infrastructure/mirk/mirk_projection.dart`** — a tiny pure function helper:
```dart
import 'dart:ui' show Offset, Size;
import '../../domain/mirk/mirk_viewport_bbox.dart';

/// Projects lat/lon to a screen offset for the current viewport + canvas size.
///
/// Linear Mercator within the viewport bbox — sufficient for the fog
/// overlay because the underlying MapLibre canvas does its own web-mercator
/// projection at the platform layer. Here we just need "put this cell
/// rectangle on the screen where MapLibre has it".
class MirkProjection {
  const MirkProjection._();

  /// Returns the screen offset for ([lat], [lon]) given the current
  /// [viewport] and canvas [size]. Offset (0,0) = top-left of canvas.
  ///
  /// Coordinates outside the viewport return offsets outside the canvas —
  /// the caller's drawing primitives (CustomPainter, Canvas clip) handle
  /// off-screen rendering natively.
  static Offset latLonToScreen({
    required double lat,
    required double lon,
    required MirkViewportBbox viewport,
    required Size size,
  }) {
    final dLat = viewport.north - viewport.south;
    final dLon = viewport.east - viewport.west;
    // guard against zero-size bbox (should never happen in prod)
    if (dLat == 0 || dLon == 0) return Offset.zero;
    final x = ((lon - viewport.west) / dLon) * size.width;
    final y = ((viewport.north - lat) / dLat) * size.height;
    return Offset(x, y);
  }
}
```

**Extract shared helper** `lib/infrastructure/mirk/tile_cell_iteration.dart`:

```dart
/// Iterates UN-revealed cells of a [VisibleMirkTile], yielding screen
/// rectangles in pixel space via [MirkProjection]. Shared across the 4
/// concrete renderers to avoid duplicating the bit-unpack loop.
///
/// Returns a `Path` containing a rectangle per unrevealed cell — cheaper
/// to `drawPath` once per tile than `drawRect` per cell.
Path buildUnrevealedCellsPath({
  required VisibleMirkTile tile,
  required MirkViewportBbox viewport,
  required Size canvasSize,
}) {
  final path = Path();
  final cellLatSpan = (tile.tileNorthLat - tile.tileSouthLat) / 64.0;
  final cellLonSpan = (tile.tileEastLon - tile.tileWestLon) / 64.0;
  for (var j = 0; j < 64; j++) {
    final cellNorthLat = tile.tileNorthLat - j * cellLatSpan;
    final cellSouthLat = tile.tileNorthLat - (j + 1) * cellLatSpan;
    for (var i = 0; i < 64; i++) {
      final bitIndex = j * 64 + i;
      final byteIndex = bitIndex >> 3;
      final bitOffset = bitIndex & 7;
      final bit = (tile.bitmap[byteIndex] >> bitOffset) & 1;
      if (bit == 1) continue;  // revealed — skip
      final cellWestLon = tile.tileWestLon + i * cellLonSpan;
      final cellEastLon = tile.tileWestLon + (i + 1) * cellLonSpan;
      final nw = MirkProjection.latLonToScreen(
          lat: cellNorthLat, lon: cellWestLon, viewport: viewport, size: canvasSize);
      final se = MirkProjection.latLonToScreen(
          lat: cellSouthLat, lon: cellEastLon, viewport: viewport, size: canvasSize);
      path.addRect(Rect.fromPoints(nw, se));
    }
  }
  return path;
}
```

```dart
class AtmosphericMirkRenderer implements MirkRenderer {
  AtmosphericMirkRenderer(this.config, {int seed = 42})
      : _noise = SimplexNoise2D(seed: seed);
  final AtmosphericConfig config;
  final SimplexNoise2D _noise;

  // Noise modulation cache keyed by (tile parentX, parentY, elapsed-second)
  // Not strictly necessary for correctness but reduces hot-path cost.

  bool _disposed = false;

  @override
  void paint(Canvas canvas, Size size, MirkPaintContext context) {
    if (_disposed) return;
    final tSec = context.frameElapsed.inMilliseconds / 1000.0;
    final baseColor = Color(config.baseColorArgb);

    for (final tile in context.visibleTiles) {
      // Sample noise once per tile — represents "local fog density multiplier"
      final noiseSample = _noise.noise2(
        tile.parentX * config.noiseScale + tSec * config.noiseSpeed * _driftX,
        tile.parentY * config.noiseScale + tSec * config.noiseSpeed * _driftY,
      );
      // Map [-1, 1] → alpha modulation ±3%
      final alpha = (config.densityBaselineAlpha + noiseSample * 0.03).clamp(0.0, 1.0);
      final argb = (baseColor.value & 0x00FFFFFF) | ((alpha * 255).toInt() << 24);

      final featherSigma =
          (size.height / 64.0) * config.featherRadiusFraction * context.pixelRatio;
      final paint = Paint()
        ..color = Color(argb)
        ..style = PaintingStyle.fill
        ..maskFilter = MaskFilter.blur(BlurStyle.inner, featherSigma);

      final path = buildUnrevealedCellsPath(
          tile: tile, viewport: context.viewportBbox, canvasSize: size);
      canvas.drawPath(path, paint);
    }
  }

  double get _driftX => math.cos(_toRad(config.driftDirectionDeg));
  double get _driftY => -math.sin(_toRad(config.driftDirectionDeg));  // screen y inverted

  static double _toRad(double deg) => deg * math.pi / 180.0;

  @override
  void update(Duration elapsed) {
    // Animation state is time-only, read from frameElapsed in paint() —
    // no per-frame mutation needed. Kept as a no-op for symmetry.
  }

  @override
  Future<void> dispose() async {
    _disposed = true;
  }
}
```

```dart
class CandlelightMirkRenderer implements MirkRenderer {
  CandlelightMirkRenderer(this.config, {int seed = 13})
      : _noise = SimplexNoise2D(seed: seed);
  final CandlelightConfig config;
  final SimplexNoise2D _noise;
  bool _disposed = false;

  @override
  void paint(Canvas canvas, Size size, MirkPaintContext context) {
    if (_disposed || context.visibleTiles.isEmpty) return;

    final tSec = context.frameElapsed.inMilliseconds / 1000.0;
    // Flicker modulation: high-freq noise
    final flicker = _noise.noise2(0.0, tSec * config.noiseSpeed * 10);  // 0.5-freq
    final alpha = (config.baselineAlpha + flicker * 0.07).clamp(0.0, 1.0);

    // Radial glow centre
    final Offset centre;
    if (context.currentFix case final fix?) {
      centre = MirkProjection.latLonToScreen(
          lat: fix.lat, lon: fix.lon,
          viewport: context.viewportBbox, size: size);
    } else {
      centre = Offset(size.width / 2, size.height / 2);
    }

    // Radial gradient shader — warm centre to dark periphery
    final glowRadius = math.max(size.width, size.height) * 0.75;
    final shader = Gradient.radial(
        centre, glowRadius,
        [Color(config.centerColorArgb).withOpacity(alpha),
         Color(config.peripheryColorArgb).withOpacity(alpha)],
        [0.0, 1.0]);

    final featherSigma =
        (size.height / 64.0) * config.featherRadiusFraction * context.pixelRatio;
    final paint = Paint()
      ..shader = shader
      ..style = PaintingStyle.fill
      ..maskFilter = MaskFilter.blur(BlurStyle.inner, featherSigma);

    for (final tile in context.visibleTiles) {
      final path = buildUnrevealedCellsPath(
          tile: tile, viewport: context.viewportBbox, canvasSize: size);
      canvas.drawPath(path, paint);
    }
  }

  @override
  void update(Duration elapsed) {}
  @override
  Future<void> dispose() async { _disposed = true; }
}
```

```dart
class HeavenlyCloudsMirkRenderer implements MirkRenderer {
  HeavenlyCloudsMirkRenderer(this.config, {int seed = 21})
      : _noise = SimplexNoise2D(seed: seed),
        _driftX = math.cos(_toRad(config.driftDirectionDeg)),
        _driftY = -math.sin(_toRad(config.driftDirectionDeg));
  final HeavenlyCloudsConfig config;
  final SimplexNoise2D _noise;
  final double _driftX, _driftY;
  bool _disposed = false;

  @override
  void paint(Canvas canvas, Size size, MirkPaintContext context) {
    if (_disposed || context.visibleTiles.isEmpty) return;
    final tSec = context.frameElapsed.inMilliseconds / 1000.0;

    for (final tile in context.visibleTiles) {
      // Coarse noise with NE drift
      final noiseSample = _noise.noise2(
        tile.parentX * config.noiseScale + tSec * config.noiseSpeed * _driftX,
        tile.parentY * config.noiseScale + tSec * config.noiseSpeed * _driftY,
      );
      final alpha = (config.baselineAlpha + noiseSample * 0.10).clamp(0.0, 1.0);
      final argb = (config.colorArgb & 0x00FFFFFF) | ((alpha * 255).toInt() << 24);

      final featherSigma = (size.height / 64.0) * 0.1 * context.pixelRatio;
      final paint = Paint()
        ..color = Color(argb)
        ..style = PaintingStyle.fill
        ..maskFilter = MaskFilter.blur(BlurStyle.inner, featherSigma);

      final path = buildUnrevealedCellsPath(
          tile: tile, viewport: context.viewportBbox, canvasSize: size);
      canvas.drawPath(path, paint);
    }
  }

  static double _toRad(double deg) => deg * math.pi / 180.0;

  @override
  void update(Duration elapsed) {}
  @override
  Future<void> dispose() async { _disposed = true; }
}
```
